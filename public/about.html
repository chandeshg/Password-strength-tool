About This Project

A solo project created to help people manage their passwords securely.

Project Overview

SecurePass is an individual project focused on password security and management. Built using modern web technologies, it provides essential tools for password generation and security checking.

Technologies Used

  • Node.js & Express
  • MySQL
  • Bcrypt Encryption
  • ZXCVBN Password Strength
